# PRD (제품 요구사항 문서)

**프로젝트명:** TSC UL타입 거푸집 측압 검토 보고서 발행
**버전:** 0.1  
**작성자(Owner):** 연구개발부 조광원 과장
**마지막 업데이트:** YYYY-MM-DD

## 1. 개요 (Overview)
- 프로젝트 배경: 2025년 연말까지 TSC UL타입 개발이 완료되어야함. 
- 해결하고자 하는 문제:
- 최종 목표:
  - TSC UL 타입 개발: 거푸집 측압 검토 -> 구조계산 시트 작성 -> 대안 설계 -> 경제성 평가 -> 접합 상세 개발 -> 구조계산 프로그램 작성

## 2. 목적 및 핵심 성능지표 (Goals / KPIs)
- 목표 1: TSC WEB에 tie 없이 가능한 TSC 높이가 몇까지인지.
- 목표 2: 용접을 얼마나 길게 어떻게 해야하는지 파악.

## 3. 세부 프로젝트
1. 콘크리트 측압 계산 (Demand)
2. 하부거푸집의 기본 정보
   1. 재료 물성치, Fy, Es
   2. 단면이차모멘트, 도심 등
3. 하부거푸집 소요하중 계산 (Demand) <- 1번을 활용하여 계산됨.
   1. 소요등분포하중 w (kN/m)
   2. L폼의 휨하중 M_max
   3. L폼의 전단하중 V_max
   4. 처짐 공식
4. 하부거푸집 강도 계산 (Capacity)
   1. L폼의 휨강도 Mn
   2. L폼의 전단강도 Vn
   3. 처짐 기준 (3mm)
5. 하부거푸집 검토 (Demand vs Capacity)
   1. L폼의 휨강도 체크 - M_max vs Mn
   2. L폼의 전단강도 체크 - V_max vs Vn
   3. L폼의 처짐 체크 - Delta_max vs delta
6. 검토방법을 알았으니, 이제는 가능한 TSC 높이 파악 - 목표 1을 위한 스터디 결과
7. 용접강도에 대한 스터디
8. 용접부위에 걸리는 소요하중 계산 (Demand) <- 1번, 3번을 활용하여 계산됨.
9. 용접부위 강도 계산 (Capacity)
10. 용접부위 검토 (Demand vs Capacity)
11. 용접 검토방법을 알았으니, 이제는 가능한 용접량(길이, 두께) 등을 산정함
